# STACK


Frontend React + Vite
Backend Deno
BD SQLITE

## Commandes

```bash
npm install
npm run dev
```

Autres commandes utiles :
- `npm run build`
- `npm run lint`

## Fonctionnement

- le frontend tourne sur `http://localhost:5173`
- les appels `/api` sont proxifies vers `http://localhost:8000`
- l'entree principale est `src/main.tsx`
- la feature principale est `src/features/infrastructure-map`
